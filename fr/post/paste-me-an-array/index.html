Coller un array

19 janv 2017
Powershell

Vous avez une liste des valuers, et vous voulez un array.

C’est possible de copier dedans un affiche et l’emporter. Ou mettre entre @" et "@ et diviser avec split ou, Create-ArrayFromPastedText.

function Create-ArrayFromPastedText ($returnvalue = "")
{
    $result = @()
    while ($true) {
        $value = read-host    
        if ($value -eq $returnvalue){
            return $result
        } else {
            $result += $value
        }
    }
}

Utiliser commme ça.

$MyArray = create-Arrayfrompastedtext
#Ici read-host coller la liste des valeurs
